操作系统基本知识

2019-12-03

简介 profile

操作系统基本知识，涉及软件运行过程，操作系统的作用，操作系统的特征，操作系统的架构等四方面。
The basic knowledgement of operating system consists of the process of software running, the functions of OS, the features of OS and the architecture of OS.

1. the process of software running

Take printing "hello world" as an example. Firstly, the OS checks whether the file is an executable file. Then when the file executing, it will generate Page Fault exception. Then the OS allocates physical memory to the program and load the program to the memory. When the program needs to print something to the screen(or other operations that need the OS to do), the OS calls other interface to finish the request.

2. the main functions of OS

There are three main functions of OS:

  1. Manage the software and hardware of the computer;
  2. Provide services or interfaces for the user;
  3. Encapsulate the underlying hardware.

3. the features of OS

Four features of the OS: concurrency, share, virtual and random.

  1. Concurrency means the capacity of handling simultaneous activities.
  2. Share means files and resources are shared.
  3. Virtual means the OS allocates resource to every process.
  4. Random means that the operating time is uncontrollable and the status of a certain moment cannot be reproduced.

4. the architecture of OS

The architecture of OS is always divided into four layers: hardware resource, operating system, DLL(dynamic linked library) and applications.
